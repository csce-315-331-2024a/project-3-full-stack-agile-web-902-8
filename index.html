Installing

Ubuntu/WSL

Since Node.js version included in Ubuntu reached EOL in 2021, we need to install the latest version differently. Windows has an article about installing Node.js, the commands summarized are

sudo apt-get update
sudo apt-get install curl
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/master/install.sh | bash # install nvm

Restart the terminal or open a new terminal,

nvm install --lts # latest stable version

Verify installation, you should see something like this about the versions

npm --version
10.5.0
node --version
v21.7.1

Once both are installed, clone the repo and install the dependencies

git clone git@github.com:csce-315-331-2024a/project-3-full-stack-agile-web-902-8.git
cd project-3-full-stack-agile-web-902-8
npm install

Running

Use the command npm run dev to run the app from the terminal. Then you can open it in a browser, the default url is localhost:3000. This will require a recent version of npm and Node.js to be installed.
